Claude Code 在处理请求时，会显示一个加载指示符，通常是 "Considering…" 或 "Analyzing…" 之类的文字，这种文字很官方，但同时也很无聊，所以我把所有的指示符文本都替换成了《飞驰人生》的引用。

> 最直接的方法是你可以让 Claude Code 帮你修改一下加载指示符的文本。告诉它你想要的主题（《火影》或者海贼王之类），它会直接更新 `~/.claude/settings.json`，本文主要介绍其工作原理，掌握了它之后你可以做更随心的切换。

## 配置项：`spinnerVerbs`

Claude Code 在 `~/.claude/settings.json` 中暴露了一个 `spinnerVerbs` 配置项，你需要设置一个模式并传递一个字符串数组，就这么简单。

```json
{
  "spinnerVerbs": {
    "mode": "replace",
    "verbs": [
      "跟张弛一起飙车...",
      "我是你的路书...",
      "激活五档...",
      "叶经理你好..."
    ]
  }
}
```

## 两种模式

- `replace`：替换所有默认文本。Claude Code 会循环使用你列表中的 verbs。
- `append`：将你的文本叠加到内置文本中。

我选择了 `replace` 模式，因为我想完全沉浸在飞驰人生的世界中，不想看到任何"Considering…"之类的默认文本。

## 如何应用

1. 打开 `~/.claude/settings.json`。
2. 在顶级任何位置添加 `spinnerVerbs` 块。
3. 重启 Claude Code。

现在每个思考阶段都会从你的列表中随机选择一条文本，尽管是个很小的定制点，但能给你的编码体验注入一些有趣的灵魂。
