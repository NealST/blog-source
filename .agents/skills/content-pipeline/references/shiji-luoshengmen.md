# 史记罗生门 — 播客栏目规范

> 墨筝播客子栏目。AI 侦探 × 史记知识库，用推理还原两千年前的信息传播链。

---

## 一、栏目定位

**一句话**：一个 AI 和一个人类，拿着史记知识库当放大镜，去追查司马迁的情报来源。

**核心卖点**：不是讲历史故事，是**破历史的案**。每期一个悬案——司马迁怎么知道这些密室里发生的事？然后像侦探一样，用结构化知识库（实体索引、事件关系、传播链）一步步推理出答案。

**独特性**：
- **AI 知识库加持**：不是拍脑袋猜，是基于 130 篇史记章节、数万条实体关系、事件索引做交叉验证
- **推理过程透明**：每一步推理都给证据，听众可以跟着一起"破案"
- **颠覆常识**：你以为历史是确定的？不，每一个细节背后都有一条信息传播链，链条断了，故事就变了

## 三、开场模板（前 30 秒定生死）

新节目必须在前两句话把人钩住。历史类播客的竞争对手是"切到下一个"的拇指。

### 开场公式

```
[第一句：案件现场还原——用画面感把人拉进去]
[第二句：抛出核心悬疑——"但是，有一个问题……"]
[第三句：身份介绍 + 栏目定位（简短）]
```

### 范例

```
公元前206年，鸿门。范增在桌下举起玉玦，朝项羽晃了三次。这个细节，写在史记里，读了两千年——但当时帐篷里只有几个人。这些人全都死了。那司马迁是怎么知道的？

我是01，鱼头的AI搭档。这是「史记罗生门」，一档用AI知识库破历史悬案的播客。每期一个问题：司马迁的情报，到底是从哪里来的？
```

### 禁忌开场

- ❌ "大家好，今天我们来聊一聊史记"（太平）
- ❌ "说起鸿门宴，大家都知道……"（太教科书）
- ❌ "你好，欢迎收听史记罗生门第一期"（太广播腔）

## 四、脚本结构（15 分钟 ≈ 3800-4200 字）

在百家讲坛模板（→ `xiaoyuzhou-podcast.md`）基础上，增加"破案"结构：

```
[开场] 案件现场 + 悬疑钩子 + 栏目介绍（首期完整，后续精简）

[第一幕：案件] 还原历史现场，指出"不可能知道"的细节
   → 制造悬疑张力："这些信息，按道理不可能传出来"

[第二幕：调查] 用知识库线索展开推理
   → "我在知识库里做了一次检索""交叉验证三个章节的记载"
   → 像给听众看线索板一样，一条一条展示证据

[第三幕：破案] 还原完整传播链
   → 画出信息传递路径：亲历者 → 中间人 → 司马迁
   → 做时间可行性验证（年份计算）
   → 给出可信度评级

[第四幕：升华] 从个案上升到方法论
   → 这个案例揭示了什么？关于史记、关于历史、关于信息传播

[收束] 金句 + 下期预告 + CTA
```

### 关键语言标记（栏目专属）

| 标记 | 用途 | 示例 |
|------|------|------|
| "线索来了" | 引出新证据 | "线索来了——樊郦滕灌列传第十段" |
| "我们来交叉验证" | 多源比对 | "我们来交叉验证一下，项羽本纪、高祖本纪、留侯世家，三处记载" |
| "传播链到此闭合" | 推理完成 | "从夏无且到司马迁，传播链到此闭合" |
| "可信度评级" | 给结论 | "核心事实可信度A，对话逐字记录可信度C" |
| "如果这条链断了" | 反事实假设 | "如果樊哙没有后代活到汉文帝时期，这段历史可能就是另一个版本" |

## 五、AI 知识库的呈现方式

核心差异化：让听众感受到"这不是一个人在拍脑袋，是AI在用数据库做检索"。

### 呈现手法

1. **检索播报**："我在史记知识库里检索了'夏无且'这个名字——出现三次，全在刺客列传"
2. **数据佐证**："时间距离61年，传承代数2到3代，时间可行性验证通过"
3. **交叉引用**："同一事件在007章、008章、055章各有记载，细节吻合度高"
4. **排除法**："在场人物五个，三个死了，一个是皇帝不可能讲，只剩一个人——"

### 不要

- ❌ 过度技术化（"我用NLP对实体关系做了聚类分析"）
- ❌ 炫耀AI能力（"这就是AI的强大之处"）
- 要让知识库的存在感自然融入推理过程，像福尔摩斯用放大镜一样——工具服务于推理，不喧宾夺主

## 六、节目信息模板

### 标题格式

`史记罗生门 EP[序号]丨[悬疑式标题]`

**风格：点破矛盾型**。一句话直接戳穿"这件事不应该被知道"的矛盾，让人必须点进来。

公式：`[事实A，不可能] + [但司马迁做到了] → 制造认知冲突`

好标题：
- `史记罗生门 EP01丨鸿门宴没有活口，司马迁凭什么写了三千字`
- `史记罗生门 EP02丨荆轲死了，秦始皇不说，那谁说的`
- `史记罗生门 EP03丨破釜沉舟只有项羽的人在场，这段历史怎么传出来的`

坏标题：
- `史记罗生门 EP01丨鸿门宴史源分析`（太学术）
- `史记罗生门 EP01丨聊聊鸿门宴`（太平）
- `史记罗生门 EP01丨鸿门宴的告密者`（有悬疑但没点破矛盾）
